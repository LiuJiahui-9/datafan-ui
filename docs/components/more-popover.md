# MorePopover 更多操作弹层
适用于一些场景不便展示多个操作时，用此组件展示更多操作。

## 基本用法

::: demo
``` html
<DMorePopover>
  <DMorePopoverItem>
    <a-button type="link">操作1</a-button>
  </DMorePopoverItem>
  <DMorePopoverItem>
    <a-button type="link">操作2</a-button>
  </DMorePopoverItem>
  <DMorePopoverItem>
    <a-button type="link">操作3</a-button>
  </DMorePopoverItem>
  <DMorePopoverItem>
    <a-button type="link">操作4</a-button>
  </DMorePopoverItem>
  <DMorePopoverItem>
    <a-button type="link">操作5</a-button>
  </DMorePopoverItem>
</DMorePopover>

```
:::
