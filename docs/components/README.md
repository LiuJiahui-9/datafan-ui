## API 

### Props

|参数|说明|类型|默认值|
|---|---|---|---|
|name|The name of the form, up to 8 characters|`String` /  `Number`|`true`|


### Events

|事件名|说明|参数|
|---|---|---|
|onclear|Fire when the form is cleared|The argument is a boolean value representing xxx|
